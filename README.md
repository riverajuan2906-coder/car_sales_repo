# Análisis de Anuncios de Venta de Coches 

## Descripción
Es un dashboard web para explorar el dataset de anuncios 
de venta de vehículos usados en EE.UU.

## Funcionalidades
- Histograma de distribución del odómetro
- Gráfico de dispersión: Precio vs Odómetro

## Tecnologías
- Python, Pandas, Plotly Express, Streamlit
