# 🚗 Análisis de Anuncios de Venta de Coches — Dashboard Interactivo

## Descripción
Dashboard web interactivo para explorar un dataset de anuncios de venta 
de vehículos usados en EE.UU. Permite visualizar patrones en el kilometraje 
y la relación entre precio y uso del vehículo.

## 🔗 App en vivo
👉 [Ver dashboard en Render](https://car-sales-repo.onrender.com/)

## Preguntas clave
- ¿Cómo se distribuye el kilometraje de los vehículos en venta?
- ¿Existe relación entre el precio y el odómetro del vehículo?

## Funcionalidades
- Histograma interactivo de distribución del odómetro
- Gráfico de dispersión: Precio vs Odómetro
- Filtros activables por checkbox

## Principales hallazgos
- La mayoría de vehículos en venta tienen entre 50.000 y 150.000 millas
- Se observa una tendencia negativa entre precio y kilometraje: 
  a mayor odómetro, menor precio
- Existen valores atípicos de precio alto con alto kilometraje, 
  posiblemente vehículos de lujo o clásicos
